Gestion Total App - BNaN

Esta aplicación web centraliza la información de clientes, empresas, acciones realizadas y por realizar en un único sistema accesible por todo el equipo de trabajo.
Permite gestionar clientes, estados, interacciones y tareas. De este modo, organiza la información de forma estructurada y accesible.
Pensada para equipos pequeños (ventas/soporte) sin herramientas formales.
